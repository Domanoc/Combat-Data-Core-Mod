[h2]ModFramework[/h2]
The [b]ModFramework[/b] is a utility mod that aims to improve mod inter-operability while giving modders tools for creating new features.

To use the [b]ModFramework[/b] it has to be first in the mod load order.

[h2]Features[/h2]
[list]
[*]Localization support.
[*]Component shop icon management.
[*]Auto fixes for many known mod bugs.
[*]Game Documentation for modders.
[*]Mod documentation for modders.
[*]Toggleable Quality of life features.
[*]And more...
[/list]

For a full list of features visit the [url=https://github.com/Domanoc/Mech-Engineer-Mods]github page[/url]

[h2]Quality of life[/h2]
[h3]Moving pilots with key combination:[/h3]
Pilots can be moved into the first available mech by clicking on the pilot in the hanger with the use of the key combination "_shift + left mouse button_". This will ignore the pilot states where you can normally not interact with them. And in reverse the pilots can be moved back into the hangar with the same key combination.

Default setting: Enabled

[h3]RTS style unit selection controls:[/h3]
By default the game unit selection method is:
- Numbers 1-8 toggles the selection of the mech with the same number.
- Shift + Numbers 1-8 will select only that mech and unselect the others.

With this feature enabled it changes to:
- Numbers 1-8 will select only that mech and unselect the others.
- Shift + Numbers 1-8 will add that mech to the current selection.
- Ctrl + Numbers 1-8 will save the current selection to the given number

Default setting: Disabled

[h3]Allow partial mech assembly[/h3]
With this feature when you don't have enough staff to complete the assembly of a mech you can instead do a partial assembly of the mech.   

Example: if you need 80 engineers to complete the assembly but only have 70 available. Then you get the option to do the work for 64 engineers. Leaving 16 engineers of work for when you do have enough.

Default setting: Disabled

[h2]Localization[/h2]
This mod offers localization support made with machine translations.
